# My Personal Portfolio

Live site: https://munyakazi.vercel.app

A modern personal portfolio built with Next.js (frontend) and Node.js/Express (backend), deployed on Vercel and Render.

## Tech Stack

- Frontend: Next.js, TypeScript, Tailwind/Shadcn (or whatever you use)
- Backend: Node.js, Express, MongoDB, Resend for emails
- Deployment: Vercel (frontend), Render (backend)

## Local Development

You need two terminals:

### Backend

```bash
cd backend
npm install
npm run dev
```
